# Shaadi Biodata Generator

A full-stack matrimonial biodata generator that allows users to create, edit, preview, and download professional biodata templates. The application provides multiple templates, privacy controls, image upload, multilingual support, and PDF export.

## Features

**User Authentication**
- Signup and login system
- Secure password hashing using bcrypt
- JWT-based authentication

**Biodata Management**
- Create biodata
- Edit biodata
- Delete biodata
- Dashboard listing all biodata

**Template System**
- Classic template
- Modern template
- Premium template

**PDF Export**
- Preview biodata before download
- Download biodata as PDF

**Privacy Control**
- Option to hide contact details
- Watermark added for free users

**Image Upload**
- Upload profile image
- Cloudinary image hosting

## Tech Stack
**Frontend**
- React
- React Router
- Tailwind CSS
- Axios
- html2pdf.js
- react-i18next

**Backend**
- Node.js
- Express.js
- JWT Authentication
- bcrypt

**Database**
- MongoDB Atlas
- Mongoose

**Cloud Services**
- Cloudinary (image storage)
